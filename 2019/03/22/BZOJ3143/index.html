BZOJ3143 「HNOI2013」游走

AcF's Blog
2019-03-22
题解
动态规划, 概率期望, 贪心, 高斯消元

Description

$n$ 个点 $m$ 条边的无向连通图，在上面从 $1$ 号点开始随机游走。现在你可以给每条边从 $1$ 到 $m$ 编号作为分数（经过就得分）。求如何编号使得总分的期望最小。输出这个最小值即可。

$n \leq 500$

Solution
