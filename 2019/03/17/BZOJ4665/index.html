BZOJ4665 小w的喜糖

AcF's Blog
2019-03-17
题解
动态规划, 容斥

Description

$n$ 颗糖发给了 $n$ 个人，每颗糖有一个种类。$n$ 个人相互交换手中的糖那么有多少种方案使得每个人手中的糖的种类都与原来不同。

两个方案不同当且仅当，存在一个人，他手中的糖的种类在两个方案中不一样。

答案对 $10^9 + 9$ 取模

答案对 $10^9 + 9$ 取模！！！！！ $10^9 + 9$ ！！！！ $10^9 + 9$ ！！！！！ $10^9 + 9$ ！！！！！

Solution
